# IBM HR Employee Attrition Analysis - SQL + Python

## Project Overview
Analyzed IBM HR Employee Attrition dataset of 
1,470 employees using SQL queries and Python 
to identify key factors driving employee turnover 
and provide actionable business insights.

## Tools Used
- Python
- Pandas
- SQL (pandasql)
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
1. Overall 16.12% of employees left the company
2. Sales department has highest attrition at 20.63%
3. Employees under 25 leave the most at 39.18%
4. Sales Representatives have highest attrition 
   by job role at 39.76%
5. Poor work life balance drives attrition at 31.25%
6. Research Directors are most loyal at only 2.50%

## SQL Queries Used
- Overall attrition rate
- Attrition by department
- Attrition by age group
- Attrition by job role
- Attrition by work life balance

IBM HR Analytics Employee Attrition Dataset from Kaggle
